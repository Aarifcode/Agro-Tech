# Agro-Tech
Agro Tech is a modern, responsive agricultural technology website designed to help farmers, agricultural businesses, and consumers access innovative farming solutions. The frontend is built using HTML, CSS, and JavaScript, providing an attractive, user-friendly, and interactive interface.
